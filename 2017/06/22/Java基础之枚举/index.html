枚举和普通类的区别

  • 枚举可以实现接口，但是枚举默认继承了java.lang.Enum类，故而不能继承其他类。
  • 枚举默认被final修饰，故而不能派生子类。
  • 枚举构造器默认使用private修饰，如果显示添加也只能使用private修饰。
  • 枚举类的所有实例必须在第一行显示列出，否则该枚举无法在被实例。这些实例默认使用public static fianl修饰。
  • 建议枚举类的成员变量都使用private final修饰。
  • 类内部的枚举默认被static final修饰。
